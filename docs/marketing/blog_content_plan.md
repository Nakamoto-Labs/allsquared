## SEO Guidelines

### On-Page Optimisation
- Primary keyword in title, URL, H1, first paragraph
- Secondary keywords in H2s and body
- 2-3% keyword density
- Internal links to related content
- External links to authoritative sources
- Meta description (150-160 characters)
- Alt text for all images

### Content Structure
- Introduction with hook (100-150 words)
- Table of contents for posts >1,500 words
- Clear H2 and H3 hierarchy
- Bullet points and numbered lists
- Pull quotes and callout boxes
- Conclusion with CTA

### Technical Requirements
- URL structure: /blog/[slug]
- Canonical tags set
- Schema markup (Article, HowTo, FAQ)
- Mobile-responsive images
- Load time <3 seconds
